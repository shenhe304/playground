# UI Tooling Experiment

A minimal web app built to compare UI output across
Stitch, Lovable, and Claude Code using the same DESIGN.md
as the single source of truth.

## Stack
- React + TypeScript
- Tailwind CSS

## Features
- Template selection
- Customization panel
- Invitation preview
- RSVP form
